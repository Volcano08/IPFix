How to Fix an Expired IP Address

By Patrick Hendrix

An expired IP Address can lead to a multitude of issues, such as loss of connectivity, an inability to access network resources, IP address conflicts, general security risks and much more. Luckily, this article is here to help you resolve your issues.

Solution 1: Release or Renew the IP Address

To release and renew the IP address in Windows:

  1. Press the Windows key and R at the same time to open the Run Dialog menu.
  2. Type ncpa.cpl and press Enter to access your network connections.
  3. Right click on the Network Connection you are using and click Properties.
  4. In the window it will open, scroll down and select Internet Protocol Version 4 (TCP/IPv4).
  5. Select Properties.
  6. In the new Properties Window, click on the General tab.
  7. Click on Obtain an IP address automatically.
  8. Hit OK twice to close out everything.

To Renew an IP Address:

  1. Follow the first four steps to release an IP Address.
  2. Click the General Tab again.
  3. Hit the Obtain an IP Address automatically.
  4. Click OK once.
  5. Close the Network Connections Window.

Solution 2: Assign a Static IP Address

To assign a static IP Address in Windows:

  1. Right-click on the network icon in the system tray and select "Open Network & Internet settings."
  2. Click on Change Adapter Options.
  3. Find the network adapter you’re using and right click it.
  4. Select properties from the menu that pops up.
  5. Select use the following IP address.
  6. Enter all required information like desired IP address, subnet mask, default gateway, and preferred DNS server.
  7. Click Ok and you’re finished.

Solution 3: Troubleshoot the Network Infrastructure

To troubleshoot the network infrastructure in Windows:

  1. Open Control Panel.
  2. Go to "Network and Sharing Center" > "Change adapter settings".
  3. Right-click on the network adapter and select "Properties".
  4. Select "Internet Protocol Version 4 (TCP/IPv4)" and click "Properties".
  5. Ensure "Obtain an IP address automatically" and "Obtain DNS server address automatically" are selected.
